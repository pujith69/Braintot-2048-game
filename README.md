# Braintot-2048-game

![CI/CD](https://github.com/pujith69/Braintot-2048-game/workflows/CI/CD/badge.svg)

A unique implementation of the classic 2048 game with YouTube video integration for an enhanced gaming experience.

## Features

- Classic 2048 gameplay mechanics
- YouTube video integration on both sides
- High score tracking using local storage
- Responsive design
- Keyboard controls (Arrow keys or WASD)

## Getting Started

### Prerequisites

- A modern web browser
- Node.js (for development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pujith69/Braintot-2048-game.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:8080`

## How to Play

- Use arrow keys or WASD to move tiles
- Combine matching numbers to create larger numbers
- Try to reach 2048!
- Press the "Start Game" button to begin
- Use "Replay" button to start a new game
